# SaaS Pricing Simulator & Invoice Generator

This project simulates how SaaS platforms calculate billing based on customer usage.

It reads customer usage data from a CSV file, applies pricing rules, and generates invoice reports.

## Features
- Supports Basic and Pro subscription plans
- Calculates usage-based billing
- Handles extra usage charges
- Generates Excel invoice reports

## Technologies
- Python
- Pandas

## How to Run

pip install -r requirements.txt  
python main.py

## Output

An Excel file is generated inside the reports folder containing invoice details.
